# Diagrams

`*.excalidraw` files are the editable source — open them at
[excalidraw.com](https://excalidraw.com) (File → Open) to edit. The matching
`*.png` is the exported render embedded in the docs; re-export (excalidraw.com →
Export image → PNG, or the research repo's `tools/excalidraw-to-png.mjs`) and
commit both whenever a diagram changes.
